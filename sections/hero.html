nexus_memory
local-first · runs fully offline

Persistent memory for
LLM agents.

Nexus is an in-process, SQLite-vec backed cognitive memory module — working, episodic, semantic and procedural memory, plus an optional hierarchical diary. One process() entry point. No server. No model download.

Python 3.11+ SQLite + sqlite-vec 5 memory layers 151 tests green MIT
Quickstart → See the architecture
agent.py 
from nexus_memory import NexusMemory

m = NexusMemory(db_path="agent.db")

# a single ingest fans out across every layer
m.process({"action": "ingest", "interaction": {
    "query":    "where do I keep my keys?",
    "response": "In the blue ceramic bowl.",
}})
m.wait()

# assemble a prompt-ready, layer-aware context
res = m.process({"action": "assemble",
                 "query": "where are my keys?"})
print(res["context_xml"])
# <memory_context> … <fact id="2">blue ceramic bowl</fact> … </memory_context>
