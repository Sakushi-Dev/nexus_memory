Quickstart

One command and you're remembering.

Clone and pip install — that's the whole setup. A virtualenv is optional. Then talk to memory through the single process() entry point, from Python or any other language.

terminal 
# clone + install — pip pulls the 3 deps and registers the package
git clone https://github.com/Sakushi-Dev/nexus_memory.git && cd nexus_memory && pip install -e .

# import nexus_memory now works anywhere (Python ≥ 3.11)
terminal 
# optional: isolate Nexus + its deps in a project-local venv
python -m venv .venv && source .venv/bin/activate   # Windows: .venv\Scripts\Activate.ps1
git clone https://github.com/Sakushi-Dev/nexus_memory.git && cd nexus_memory && pip install -e .
terminal 
# the defaults need nothing extra. opt into a richer embedder:
pip install -e ".[sentence-transformers]"   # local transformer embeddings
pip install -e ".[openai]"                  # hosted embeddings

pip install -e ".[dev]" && python -m pytest -q   # run the offline tests · 159 passed ✓

Talk to it

Plain JSON in, plain JSON out.

Ingest an instruction once; it returns as a directive on every assemble and persists across sessions.

behavior.py 
from nexus_memory import NexusMemory
m = NexusMemory()

# a standing instruction is detected + persisted automatically
m.process({"action": "ingest", "interaction": {
    "query":    "Always respond in German.",
    "response": "Got it.",
}})
m.wait()

# it returns as a directive on every assemble — inject it
# into your system prompt and the behavior persists.
ctx = m.process({"action": "assemble", "query": "hi"})
print(ctx["directives"])      # ['Respond in German.']
bridge.py 
# a ten-line bridge: one JSON request per line in → one JSON response out
import sys, json
from nexus_memory import NexusMemory

m = NexusMemory(db_path="agent.db")
for line in sys.stdin:
    print(json.dumps(m.process(json.loads(line))), flush=True)
client.js — Node.js 
// any language can drive it — here Node over stdin/stdout
import { spawn } from "node:child_process";
const py = spawn("python", ["bridge.py"]);

py.stdout.on("data", b => console.log(JSON.parse(b).context_xml));
py.stdin.write(JSON.stringify({ action: "assemble", query: "where are my keys?" }) + "\n");

process() actions — one verb per call

assembleBuild a <memory_context> for a query.
ingestStore one exchange; fans out across layers.
inspectRead store health or any layer.
forgetDelete by id or free-text match.
optimizeCompact — VACUUM + WAL checkpoint.
diaryDay-summary or transcript reconstruction.
ruleAdd / list / deactivate directives.
distillPromote hot facts into rules.
pending_summariesDrain the diary outbox.V
submit_summaryFold a summary back in.V
