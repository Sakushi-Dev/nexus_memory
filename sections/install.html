Quickstart

Two files and you're remembering.

Install into a project-local virtualenv, then talk to memory through process() — or wire the standing behavior straight into your system prompt.

install.sh 
# a project-local venv, editable install
python -m venv .venv
./.venv/Scripts/python.exe -m pip install -e .

# run the offline self-test — no API key needed
./.venv/Scripts/python.exe -m pytest -q
# 151 passed ✓
behavior.py 
# a standing request is detected and persisted…
m.process({"action": "ingest", "interaction": {
    "query":    "Sprich ab jetzt deutsch mit mir.",
    "response": "Alles klar.",
}})
m.wait()

res = m.process({"action": "assemble", "query": "hi"})
assert "Respond in German." in res["directives"]
# …inject directives into the system prompt — behavior persists.
