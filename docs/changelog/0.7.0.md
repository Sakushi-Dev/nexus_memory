# 0.7.0 — 2026-06-21

An opt-in **local semantic embedder** so paraphrased recall works — without
coupling the module to any vendor. The default `HashingEmbedder` is lexical
(bag-of-words): it only matches on shared words, so a question like *"what's my
project's name?"* misses the stored fact *"I'm building a tool called Tideglass"*.
0.7.0 adds a real semantic embedder that runs **locally** (downloads a small model
once, then fully offline) — the module stays provider-agnostic, no API required.

## Added

- **`FastEmbedEmbedder`** — built on `fastembed` (ONNX Runtime, **no PyTorch**),
  default model `BAAI/bge-base-en-v1.5` (**768-dim — same as the default**, so no
  schema change). Downloads ~210 MB once, then runs offline on CPU. Opt-in:
  `pip install "nexus-memory[local-embeddings]"`.
- **Declarative selection:** `NexusConfig(embedder_backend="fastembed")` plus
  `embedder_model` / `embedder_cache_dir` / `embedder_offline` knobs (an explicit
  `embedder=` injection still wins). `HashingEmbedder` remains the zero-dependency
  default; a bare install pulls no neural dependency.
- **Safety:** a dim guard (checked against the real stored vector dimension) and
  embedder **provenance** recorded in `system_config` — opening a store with a
  different embedder than it was written with is refused, never silently mixed.
- **Re-index tool:** `python -m nexus_memory.reindex --db <path> --backend fastembed`
  re-embeds an existing store; the same-dimension path (hashing 768 → bge-base 768)
  runs in a single transaction with no schema change.

## Compatibility

- Zero SQL DDL; no public API removed. The default behavior is unchanged
  (`HashingEmbedder`). Switching embedder requires a one-time re-embed (the tool
  above); the store refuses to mix vector spaces silently.

## Verification

- Full suite green (**237 passed**, +6 new incl. the semantic-vs-lexical recall
  regression). Verified end-to-end in the demo (real chat + the local embedder):
  the paraphrased *"what is the exact name of my project and which language?"* now
  retrieves the Tideglass fact (which the lexical default scored 0.0 / missed).
